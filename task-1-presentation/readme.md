# Task 1 – Digital Literacy Infographic

In this task, I designed an infographic to raise awareness about digital literacy. I have designed it using the online infographic creator, Canva. The infographic describes the concept of digital literacy and focuses on important aspects such as safe internet practices, digital tools for students, professional online presence, and email etiquette.

The main goal of creating the infographic is to provide essential digital knowledge in a simple and easy-to-understand way for students.

The final infographic is included in the folder as an image file.
